◆ GameWindow_init()

void GameWindow_init ( float gw_screen_width,
float gw_screen_height )

init function for the game window Global variables are used to avoid having to pass window sizes to every function and to avoid updates of values when no changes were made to the window. This also avoids the deallocation and reallocation of the same size of memory for variables when the function gets called again for retry. Variables used for the game logic are initialized every time the function is called. Variables of which the values are tied to the screen size are only initialized if the screen size has changed from the last call. This is done through the global variables screen_width and screen_height, which store the size of the previous window and are compared to the passed values.

NOTE: Buttons are stored in an array. Current values: ButtonArray[0] = hit, ButtonArray[1] = stand, ButtonArray[2] = surrend, ButtonArray[3] = double down, ButtonArray[4] = bet this is done to facilitate adding new buttons if needed for modifications to the game.
